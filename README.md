## Overview

Lint code using LLMs.

## Getting Started

```bash
usage: LLinter [-h] [-r RULES] [-p PROMPT] [-m MODEL] script

Lints a target file using LLMs.

positional arguments:
  script

options:
  -h, --help            show this help message and exit
  -r RULES, --rules RULES
  -p PROMPT, --prompt PROMPT
  -m MODEL, --model MODEL
```
